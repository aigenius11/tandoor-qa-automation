# Отчет о найденных дефектах (Bug Reports)

### [BUG-01] Ошибка 403 Forbidden / Connection Refused при импорте рецептов через API
* Приоритет: Критический (Blocker)
* Компонент: API / Import Module
* Описание: При попытке импорта рецепта с внешних ресурсов через API-запрос сервер возвращает ошибку 403 или сбрасывает соединение. Это блокирует генерацию ID рецептов, необходимых для дальнейших этапов автоматизации.
* Ожидаемый результат: Сервер принимает внешнюю ссылку, успешно импортирует рецепт и возвращает валидный JSON с ID нового рецепта.
* Фактический результат: ConnectionRefusedError или HTTP 403 Forbidden. Дальнейшие тесты, зависящие от ID рецепта, не могут быть выполнены.

### [BUG-02] Невозможность выбора даты в календаре и взаимодействия с Meal Plan (UI)
* Приоритет: Высокий (High)
* Компонент: Frontend / Meal Plan / Calendar
* Описание: В сценарии планирования питания процесс блокируется на начальном этапе — выборе даты в интерактивном календаре. Несмотря на использование точных локаторов и путей, клик по дате не инициирует открытие формы создания приема пищи. Это делает невозможным выполнение всего сценария Meal Plan.
* Ожидаемый результат: При нажатии на конкретную дату в календаре происходит активация события и открывается модальное окно/карточка для ввода данных.
* Фактический результат: Элементы дат в календаре найдены драйвером, но программный клик не вызывает ответной реакции интерфейса. Дальнейшие шаги сценария блокируются.

### [BUG-03] Ошибка заполнения карточки рецепта / Неактивные поля ввода (UI)
* Приоритет: Высокий (High)
* Компонент: Frontend / Recipe Creation
* Описание: При попытке автоматизированного заполнения полей в карточке создания рецепта, система не фиксирует ввод данных или не дает нажать на кнопки сохранения. 
* Ожидаемый результат: Все поля ввода доступны для взаимодействия, данные успешно сохраняются в профиле.
* Фактический результат: Попытки взаимодействия с полями ввода через стандартные инструменты автоматизации не дают результата, кнопки сохранения не реагируют на программные клики.
